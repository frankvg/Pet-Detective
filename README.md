# Pet-Detective
Pet Detective monitors pet movement in and out of a pet door, records the action, and logs the time. 
It uses an ESP32 processor with a Time-of-Flight laser sensor.
The software is mostly C++ for Arduino, along with some HTML and JavaScript.
See the PDF file for full details and additional software development notes.
